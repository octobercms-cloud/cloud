# `octobercloud/cloud`

Welcome to the official runtime and CLI for [October Cloud](https://octobercms.cloud)

## Getting started

This project is currently under heavy development, you may want to check back later.

For those who are brave, run the following.

```bash
composer require octobercloud/cloud --dev

php artisan cloud:install
```
